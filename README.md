# thermdrag

## Personal Starter / learning project

* Purescript
* Webpack 2
* Hot Code Reloading with Purs-loader 
* Thermite

_Testing some SVG drag and drop_
_And other stuff_
_Im using Visual Studio Code and Purescript 0.11.4 built from source, but 0.11.3 also worked fine_

* ./start

#### Sample Deployment
* ./build
